# Capturas del MVP

## Objetivo
Registrar evidencia visual del funcionamiento del sistema en sus principales etapas.

## Capturas recomendadas
- login.png: pantalla de acceso al sistema.
- dashboard.png: vista principal con intervenciones registradas.
- nueva-intervencion.png: formulario de captura.
- registro-guardado.png: confirmación de una intervención almacenada.
- resumen-ia.png: vista de síntesis asistida por IA.
- voz.png: ejemplo del uso de dictado por voz.

## Recomendación
Se sugiere almacenar imágenes en formato PNG con nombre claro y fecha o versión cuando sea posible.
