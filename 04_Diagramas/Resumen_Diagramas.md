# Resumen de diagramas del proyecto

## 1. Propósito
Esta carpeta reúne los principales diagramas que representan el funcionamiento del sistema de registro inteligente de intervenciones.

## 2. Diagramas incluidos
- Casos_Uso.png: representa los usuarios y las acciones que pueden realizar.
- Flujo_Proceso.png: describe el flujo operativo del registro y seguimiento.
- Modelo_ER.png: muestra la estructura de datos del sistema.
- Roles_Permisos.png: ilustra los roles y permisos esperados.
- Secuencia_Registro.png: muestra la interacción entre componentes durante el registro.

## 3. Interpretación breve
- El usuario inicia sesión y accede al formulario.
- Captura la intervención mediante texto o voz.
- El sistema valida y almacena la información.
- El dashboard permite consultar y resumir registros.

## 4. Valor para la documentación
Estos diagramas ayudan a comunicar el proyecto de forma visual a usuarios, directivos y equipos técnicos.
