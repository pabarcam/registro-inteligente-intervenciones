# Plantilla de diagrama

## Objetivo
Describir visualmente un proceso, flujo o estructura del sistema.

## Secciones sugeridas
- actores o usuarios,
- procesos principales,
- base de datos o almacenamiento,
- decisiones o validaciones,
- resultados esperados.

## Recomendación
Cuando se actualice el proyecto, conservar el mismo estilo visual para facilitar la comparación entre versiones.
